- **Para quem é este conteúdo**: Qualquer profissional de tecnologia que queira entender recursos avançados do GitHub Copilot aplicados ao trabalho com Python e dados. Software Engineers, Data Engineers, Data Scientists e qualquer pessoa que trabalhe com Python ou dados também pode se beneficiar deste workshop.
- **O que você vai aprender**: Você vai usar recursos avançados do GitHub Copilot e aplicá-los a um projeto com Python e dados. Você aprenderá como fazer perguntas sobre o projeto e aplicar esse conhecimento em outros projetos de dados e programação.
- **O que você vai construir**: Uma ferramenta de linha de comando que usa Python e transforma um conjunto de dados CSV para que ele possa ser usado em um data pipeline ou em um ambiente de CI/CD.

## <a id="learning-objectives"></a>Objetivos de Aprendizado

Neste workshop, você irá:

  - Usar recursos avançados do GitHub Copilot, como slash commands, inline chat e agents como `@workspace`
  - Aplicar sugestões e interações do GitHub Copilot em um projeto de dados
  - Entender como construir uma ferramenta útil para trabalhar com dados em um ambiente de CI/CD ou data pipeline

## <a id="mega-prerequisites"></a>:mega: Pré-requisitos

Antes de participar do workshop, existe apenas um pré-requisito: você precisa ter uma conta pública no GitHub. Todos os recursos, dependências e dados fazem parte do próprio repositório.
